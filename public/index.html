Welcome to gmail04

A nostalgic approach to email.

gmail04 is an experiment in a new (not really) kind of webmail, built on the idea that you should never have to change your muscle memory and you should always be able to find the message you want. The key features are:

  • Search, don't sort.
    Use regular search to find the exact message you want, no matter when it was sent or received.
  • Don't throw anything away.
    However much your computers has of free storage so you'll never need to delete another message.
  • Keep it all in context.
    Each message is grouped with all its replies and displayed as a conversation.
  • No ads at all.
    You see only links to related Google web pages for accuracy.
